# Refurbished Marketplace ♻️

A simple and beginner-friendly web application where users can sell used products, admins can refurbish and certify them, and buyers can purchase trusted refurbished products.

This project demonstrates a basic marketplace workflow using frontend technologies and browser LocalStorage.

---

# 🌟 Project Overview

The Refurbished Marketplace connects:

- **Sellers** → Upload old or used products
- **Admins** → Review, refurbish, and certify products
- **Buyers** → Purchase certified refurbished products

---

# ✨ Features

## 👤 Seller Module
- Upload product details
- Add product image
- Submit products for admin approval

## 🛠️ Admin Module
- View uploaded products
- Approve and refurbish products
- Set product price
- Add warranty information
- Add product demo video link
- Mark products as certified
- Publish products to buyer store

## 🛒 Buyer Module
- View certified refurbished products
- Check pricing and warranty
- Watch product demo videos
- Browse available refurbished items
